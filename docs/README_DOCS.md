## 3. システム概要

### 3.1 システム名
Tri-force Koenji施設予約システム

### 3.2 システムの目的
会員向けの施設予約管理システムを提供し、効率的な施設利用を実現

### 3.3 主要機能
- **予約管理**: Webフォームからの予約受付
- **重複チェック**: 定員制限による予約制御
- **データ管理**: スプレッドシートでのデータ管理
- **スケジュール管理**: Googleカレンダーとの連携

### 3.4 技術スタック
- **フロントエンド**: HTML5, CSS3, JavaScript, Materialize CSS
- **バックエンド**: Google Apps Script
- **データベース**: Google Spreadsheet
- **カレンダー**: Google Calendar
- **テスト**: カスタムテストフレームワーク

## 5. 開発環境セットアップ

### 5.1 前提条件
- Node.js 12以上
- Google Apps Script アカウント
- Google Drive アクセス権限

### 5.2 セットアップ手順
1. リポジトリクローン
2. 依存関係インストール: `npm install`
3. Google Apps Script プロジェクト作成
4. 設定ファイル（.clasp.json）の設定
5. スクリプトプロパティの設定

### 5.3 開発コマンド
```bash
npm run clean      # ビルドファイル削除
npm run lint       # コード品質チェック
npm run build      # ビルド実行
npm run test       # テスト実行
npm run deploy     # 開発環境デプロイ
npm run deploy:prod # 本番環境デプロイ
```
