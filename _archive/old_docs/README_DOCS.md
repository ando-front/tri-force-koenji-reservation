## 📊 システム概要

### 現在のシステム構成

- **フロントエンド**: HTML + Materialize CSS + JavaScript
- **バックエンド**: Google Apps Script (JavaScript)
- **データストレージ**: Google Spreadsheet
- **カレンダー連携**: Google Calendar API
- **認証**: Googleアカウント認証

### 主要機能

1. **予約フォーム**: 施設・日時・利用者情報の入力
2. **重複チェック**: 既存予約との重複確認
3. **データ管理**: スプレッドシートへの予約データ保存
4. **カレンダー連携**: Googleカレンダーへの予約登録
5. **エラー処理**: バリデーション・エラーハンドリング

### 既知の制約・課題

- **GAS複数アカウント問題**: 複数Googleアカウントログイン時のアクセス不可
- **スケーラビリティ**: Google Apps Scriptの実行時間制限
- **カスタマイズ性**: GASプラットフォーム固有の制約
- **セキュリティ**: 認証・権限管理の制限
