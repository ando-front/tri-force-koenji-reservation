# Tri-force Koenji予約システム - GitHub Pages版

Google Apps Scriptで作成された予約システムをGitHub Pagesに移行したバージョンです。

## 機能

- フリーマット/フィットネス施設の予約
- 日時指定予約
- 予約状況確認
- カレンダー連携

## 構成

- **フロントエンド**: GitHub Pages上でホスティングされるHTML/CSS/JavaScript
- **バックエンド**: Google Apps ScriptのWebアプリケーションをAPIとして利用
